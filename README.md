# AI Agent Trust Infrastructure Tool Use

This example demonstrates how an AI agent can build a trust infrastructure by delegating critical, verifiable tasks to specialized, reliable tools instead of solely relying on an LLM's internal knowledge. It simulates an LLM recognizing the need for a definitive answer and then invoking a trusted 'prime number checker' tool for accuracy, ensuring the agent's response is reliable.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
